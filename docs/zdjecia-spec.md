# Spec zdjęć — Detailing Łącko

Obecne grafiki w `apps/strona/public/images/` to **art-directed placeholdery**
(generowane skryptem `pnpm generate:images`). Przed startem kampanii podmień je
na prawdziwe fotografie — wgrywasz je w panelu: **Magazyn → Galeria przed/po**
(hero: **Magazyn → CMS → Strona główna**).

## Zasady wspólne (spójna „gradacja")

- Telefon wystarczy (dobre światło > sprzęt). Zawsze **ten sam kadr** przed i po
  — statyw albo oparcie telefonu o coś stałego.
- Światło dzienne rozproszone (cień, brama garażu). Nie używać lampy błyskowej.
- Format poziomy 4:3, minimum 1600×1200 px. Bez filtrów Instagram — jedna
  wspólna korekta: +10 kontrast, -5 nasycenie, lekko chłodny balans bieli.
- Tło uporządkowane: żadnych wiader/butelek w kadrze „po".

## Lista ujęć (6–8 par, priorytet od góry — brief)

| # | Ujęcie | Uwagi |
|---|--------|-------|
| 1 | **Kanapa tylna** przed/po praniu | Klika się najlepiej (brief: tło hero). Kadr prosto z boku, całe siedzisko. |
| 2 | **Fotel kierowcy** przed/po | Plamy z kawy/jedzenia. Kadr 3/4 z progu drzwi. |
| 3 | **Reflektor** przed/po polerowaniu | Zbliżenie, odbicie nieba pokazuje przejrzystość. |
| 4 | **Maska/drzwi** przed/po one step | Pod kątem ~30° do słońca — rysy widoczne w „przed". |
| 5 | **Wykładzina/dywaniki** przed/po | Błoto po zimie. Kadr z góry. |
| 6 | **Felga** przed/po | Pył z klocków. Zbliżenie na całą felgę. |
| 7 | (zapas) Podsufitka przed/po | |
| 8 | (zapas) Skóra — kierownica/fotel przed/po | |

## Hero

Najlepsza para nr 1 (kanapa) — wersja „po" jako tło hero (podmiana w
Magazyn → CMS → Strona główna → zdjęcie desktop/mobile). Zdjęcie poziome,
minimum 2400 px szerokości, punkt ostrości w prawej połowie kadru
(tekst strony stoi po lewej).

## Alt-teksty (SEO)

Wzór z briefu: `pranie tapicerki Łącko — fotel przed i po` — pole „Alt (SEO)"
w edytorze galerii podpowiada format.
